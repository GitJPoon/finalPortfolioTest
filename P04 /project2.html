GrapeVine Interface Design

Year: January 2014 - March 2014

Role: Designer, Prototyper

Type: Solo Project

Design Concept

The idea of Dineout can be confusing to some people that have never heard of it or are new to the event. Addtionally, since there is currently no application that exist in assisting users with the Dine Out event, our application concept aims to achieve at solving this problem. The purpose of this application will help users e ectively gather various restaurant information, as well as help organize and make reservations at a much smoother and efficient pace.
